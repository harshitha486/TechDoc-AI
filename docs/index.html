Features

Everything your docs need

Four powerful tools in one app — no setup beyond an API key.

📝
Code → Docs
Paste or upload source files. Get docstrings, a README, API reference, and architecture overview in seconds.
🔀
PR Analyzer
Point to any GitHub pull request URL. Get a professional changelog entry following Keep a Changelog format.
🚀
Deployment Docs
Paste deployment diffs, commit messages, or config changes. Generate release notes and migration guides automatically.
🌐
API Documentation
Paste Flask/FastAPI/Express routes or upload an OpenAPI JSON spec. Get a full, navigable API reference.
🔄
Diff-Aware Updates
Paste old + new code. The agent updates only what changed — your existing docs stay intact.
📦
Multi-Format Export
Download generated docs as Markdown, styled HTML, or a ZIP archive containing all outputs at once.

Supported Languages

Python JavaScript TypeScript Java Go Rust C / C++ Ruby PHP C#

How it works

From code to docs in 4 steps

1
Choose a provider
Gemini and Groq are free. Anthropic Claude is paid. All are available in the sidebar.
2
Enter your API key
Keys are never stored — they live only in your browser session.
3
Paste code or URL
Pick a tab (Code, PR, Deploy, API), add your source, choose output types.
4
Download your docs
Export as Markdown, styled HTML, or a complete ZIP archive.

Start documenting in 30 seconds

No sign-up. No install. Just paste your API key and go.
